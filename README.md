# Description

- OS : MacOS 12.5(Monterey)
- python : 3.8.9

# venv

## create `.venv`

```
$ python3 -m venv .venv
```

## activate `.venv`

```
$ source ./.venv/bin/activate
```

## install dependencies

```
$ pip install -r requirements.txt
```

## export dependencies

```
$ pip freeze > requirements.txt
```
